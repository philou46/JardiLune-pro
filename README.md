# 🌙 JardiLune Pro

**JardiLune Pro** est un logiciel de jardinage basé sur le calendrier lunaire.

Il permet d’aider les jardiniers à organiser leurs travaux au jardin selon les périodes favorables de la lune : semis, plantations, tailles, récoltes et entretien.

## ✨ Fonctionnalités

- Consultation du calendrier lunaire
- Conseils de jardinage selon les périodes
- Organisation des travaux au jardin
- Interface simple et claire
- Logiciel pensé pour un usage familial et pratique

## 💻 Installation

Téléchargez la dernière version depuis la page **Releases** :

➡️ Cliquez sur le fichier d’installation `.exe`, puis lancez-le sur votre ordinateur Windows.

## 🪴 Pour qui ?

JardiLune Pro s’adresse aux jardiniers amateurs qui souhaitent planifier leurs travaux de manière simple, naturelle et organisée.
